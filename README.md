# Spam–Ham Classifier

**Tech:** Python, NumPy, pandas  
**Model:** Naive Bayes (implemented from scratch)  
**Purpose:** Simple, explainable text classifier built from first principles (tokenization, frequency counts, Laplace smoothing).

## Features
- Implemented tokenization, vocabulary building and probability estimation without ML libraries.
- Trains on labelled text and evaluates using confusion matrix, accuracy, precision/recall.
- Minimal dependencies — easy to read and extend for learning.
